# Face-Based Student Attendance System 2.0

This project is a facial recognition-based attendance management system built with Django.  
It leverages **facenet-pytorch** for face detection and recognition, and **OpenCV** for image processing.  

### Features
- 🔐 Secure user onboarding with hashed passwords
- 📸 Face recognition for automated attendance
- 🗂️ Attendance records stored in SQLite database
- 🌐 Web-based interface built with Django
- ⚡ Easy setup with `requirements.txt`
